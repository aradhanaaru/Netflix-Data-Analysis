# Netflix Data Analysis

## Overview

This project analyzes 8807 Netflix titles using Python, Pandas, Matplotlib, and Seaborn.

The goal is to identify trends in Netflix content such as:
- Movies vs TV Shows
- Content distribution by country
- Genre analysis
- Ratings analysis
- Growth of content over time

## Dataset

Netflix Titles Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed

- Data Cleaning
- Missing Value Handling
- Content Type Analysis
- Country-wise Analysis
- Genre Analysis
- Rating Analysis

## Key Insights

- Movies are more common than TV Shows.
- Some countries contribute significantly more content.
- Netflix content has grown rapidly over recent years.

- Data Cleaning
- Exploratory Data Analysis
